Flow 04 — Deadline Reminder

Runs daily at 9 AM Singapore time. Checks for RFPs with deadlines within 7 days, identifies partners who have not responded, and sends courtesy or final reminders via email. Posts a summary to Teams.

SharePoint Office 365 Outlook Microsoft Teams
Daily 9AM
→
Init Count
→
Get Active RFPs
→
Calc Days Left
→
If ≤7 days
→
Email Partners
→
Teams Summary
1

Create a Scheduled Cloud Flow

In Power Automate > + Create > Scheduled cloud flow. Name: RFP Deadline Reminder - Daily Check. Set to run every 1 day starting at 09:00, timezone Singapore Standard Time.

2

Initialize Variable Variable

reminderCount — Integer — Value: 0

3

Get Items — Active RFPs SharePoint

FieldValue
List NameRFP Tracking
Filter QueryStatus eq 'Issued' or Status eq 'Collecting'
4

Apply to Each Active RFP Control

Inside the loop, add these actions:

  1. Compose — Calculate days until deadline:
    Expression: div(sub(ticks(items('Apply_to_each')?['SubmissionDeadline']),ticks(utcNow())),864000000000)
  2. Condition — daysUntilDeadline is less than or equal to 7

    If Yes:

    1. Compose — Set reminder type: if(lessOrEquals(daysUntilDeadline, 2), 'FINAL', 'COURTESY')
    2. Get items — Partner Tracking, filter: RFPReference eq '[Title]' and ResponseReceived eq 0
    3. Apply to each pending partner:
      • Send email (V2): Subject = "[FINAL/COURTESY] REMINDER: [RFP Ref] — Response Due [Deadline]". Set Importance to High for FINAL reminders.
      • Update item on Partner Tracking: increment RemindersSent by 1
      • Increment variable: reminderCount + 1
5

Condition: Post Summary Control + Teams

After the loop, check if reminderCount > 0. If yes, post to Teams: "Daily Reminder Summary — Reminders sent: [count] — Date: [today]".

Flow 06 — Post-Deadline Summary

Runs daily at 10 AM SGT. Finds RFPs whose submission deadline has passed, counts responses received vs pending, updates status to "Collecting", and posts a summary card to Teams.

SharePoint Microsoft Teams
Daily 10AM
→
Get Past-Due RFPs
→
For Each RFP
→
Count Responses
→
Update Status
→
Teams Summary
1

Create a Scheduled Cloud Flow

In Power Automate > + Create > Scheduled cloud flow. Name: Post-Deadline Summary - Daily Check. Run every 1 day at 10:00, timezone Singapore Standard Time.

2

Get Items — Past-Deadline RFPs SharePoint

FieldValue
List NameRFP Tracking
Filter QueryStatus eq 'Issued' and SubmissionDeadline lt '[utcNow()]'
Expression for the date: formatDateTime(utcNow(),'yyyy-MM-ddTHH:mm:ssZ')
3

Apply to Each Past-Deadline RFP Control

Inside the loop, add these actions in sequence:

  1. Get items — Partner Tracking list, filter: RFPReference eq '[current RFP Title]'

    This returns all partners for this RFP.

  2. Filter array — (Data Operation) Filter the partners where ResponseReceived equals true

    This gives you the count of received responses.

  3. Compose — Calculate summary:
    ValueExpression
    Total partnerslength(body('Get_Partners')?['value'])
    Receivedlength(body('Filter_Received'))
    Pendingsub(total, received)
  4. Update item — RFP Tracking: set Status = Collecting
  5. Post message in channel — Teams:
    Message:
    Submission Deadline Passed: [RFP Reference]
    Deadline was: [SubmissionDeadline]
    Total partners invited: [total]
    Responses received: [received]
    Responses pending: [pending]
    If pending = 0: "All responses received! Ready for evaluation."
    If pending > 0: "Some responses are still pending."
